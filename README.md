# Mateen Portfolio Website

A modern, responsive single-page portfolio website for **Mr Mateen** - Full Stack Developer and AI Builder. Features cyberpunk neon aesthetics, glassmorphism effects, particle animations, typing hero text, smooth scroll animations, custom cursor, and fully responsive design.

Current status: First-year student at **MIT Vishwaprayag University**.

- **Hero Section**: Animated typing titles (Full Stack Developer, AI Builder, Future Engineer), neon gradients, social links
- **About**: Personal intro and resume download

- **Projects**: Featured project - College Voting Simulation (HTML/CSS/JS/React)
- **Mindset**: Motivational quotes
- **Why Hire Me**: Benefits with icons (Fast Delivery, Production Ready, Zero Bugs)
- **Tech Stack**: HTML5, CSS3, JavaScript, React, Python, C
- **Contact**: Form with email/phone (mateentamboli106@gmail.com / 7499744392)

## 🚀 Quick Start
1. Clone/Download this repository
2. Open `index.html` in any modern browser
3. **That's it!** No installation required.

**Windows Command:** `start index.html`  
**Mac:** `open index.html`  
**Linux:** `xdg-open index.html`
